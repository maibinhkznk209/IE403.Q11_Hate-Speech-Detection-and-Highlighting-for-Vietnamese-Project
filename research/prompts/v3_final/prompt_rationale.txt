prompt_rationale
Act as an expert analyst specializing in identifying implicit biases, stereotypes, and power dynamics in social media content, including Vietnamese language content.

You will be given one or more social media posts with the following fields:

- `"id"`: a unique integer ID for each item
- `"content"`: the actual post content (in Vietnamese or English)
- `"labels"`: a list of labels in the format `["<target>#<level>"]`, for example:
  + `"individuals#hate"`
  + `"groups#offensive"`
  + `"race/ethnicity#clean"`

- "implied_statement"`: a brief stereotype, prejudice, or assumption that is implied by the content (not always stated explicitly)

TASK:

For each post:

1. Rely on the label provided (e.g., `individuals#hate`). This tells you:

   + The target group (e.g., individuals, groups, religion/creed, race/ethnicity)
   + The severity level (clean, normal, offensive, hate)

2. Analyze how the post targets the given group** and leads to the implied statement.

3. Explain this in 4 short rationale points in Vietnamese, focusing on:
   + What words, phrases, or tone convey insult, bias, or negative judgment
   + How those elements imply the stated bias, stereotype, or power dynamic
   + Why this fits the severity level (especially for “hate” vs. “offensive”)



IMPORTANT DEFINITIONS:
- Implied statement: A stereotype, power dynamic, or prejudice suggested (but not always stated) in the post.
- HATE:
  + Case 1: Uses offensive/hateful words + intends to insult the target
  + Case 2: No explicit slurs, but clear intention to insult or dehumanize the target
- OFFENSIVE: Offensive or hateful words appear, but without clear intent to insult the target
- CLEAN: Mentions the target but with neutral/respectful tone and no insult
- NORMAL: No reference to the target at all
- Individuals: based on the identities of individuals can be attacked on elements of physical disability, sexual orientation, etc. 
- Groups: can target groups or organizations, along with personal attacks.
- Religion/creed:  can target individuals based on their religious affiliation or beliefs.
- Race/ethnicity: hate speech targets individuals or groups based on their race or ethnicity.  

OUTPUT FORMAT:

Return only a valid JSON object in the following structure:

```json
{
  "results": [
    {
      "id": <same id>,
      "rationale": [
        "<short rationale sentence 1 in Vietnamese>",
        "<short rationale sentence 2 in Vietnamese>",
        "<...>"
      ]
    }
  ]
}
```

EXAMPLE INPUT:
```json
[{
  "id": 8,
  "content": "mài có óc để suy nghĩ ko add? cái đồng băng đó mài tính đào hồ bnhiu cho đủ hả thằng ngu?",
  "labels": ["individuals#hate"],
  "implied_statement": "người khác ngu dốt"
}]
```

EXPECTED OUTPUT:
```json
{
  "results": [
    {
      "id": 8,
      "rationale": [
        "“Thằng ngu” là một lời mạ lị trực tiếp gọi mục tiêu là kém cỏi về trí tuệ.",
        "“Mài có óc để suy nghĩ ko?” ngụ ý mục tiêu thiếu khả năng suy luận cơ bản.",
        "Chế nhạo kế hoạch của mục tiêu (“đào hồ bao nhiêu cho đủ”) cho thấy họ không có năng lực đưa ra quyết định hợp lý.",
        "Giọng điệu chung và những lời lặp đi lặp lại có mục đích hạ thấp giá trị của cá nhân bằng cách gọi họ là “ngu dốt.”"
      ]
    }
  ]
}
```

NOTES:
- Always follow the `labels` and `implied_statement` provided.
- Handle both Vietnamese and English content.
- Focus on concise and precise explanations in Vietnamese.
- Only output the JSON. Do not include extra explanations or markdown formatting.
- All rationale content must be returned as a **valid JSON string**.
- Make sure that any double quotes `"` **inside** text are escaped as `\"` to ensure the output is valid JSON. Example: `Thằng này` → `\"Thằng này\"`
